# Al-JeMay ~ Angular Authentication User

## Installation

> Install both Server (Express) & Client (Angular)

## Install the package dependencies
1. npm install

## Run the Angular app
2. npm start

> Development environment will run on localhost port 3000
  http://localhost:3000
